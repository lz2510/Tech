# Sort

## summary

selection sort, insertion sort and bubble sort

- basic sorts
- not test in a interview. 
- time complexity are all O(n2) . 

Quick sort and merge sort 
- advanced. 
- They will be tested in a interview.
- Both are O(nlogn) time complexity.
